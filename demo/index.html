Embedded ALA Comments

Thanks to Mat Marquis, we'll soon be able to offer the ability to embed A List Apart comments anywhere.

The embed code consists of semantic code representing the comment (as a fallback) and a script tag that calls a live comment from our server. At the moment the semantic code is identical to the code that makes up a comment on the site; we might want to simplify it.
